# NKAcademy

NKAcademy is a mod for Elin that adds a school-themed experience and related systems.

The mod introduces original zones, guilds, quests, characters, dramas, and various supporting systems.

This repository contains the source code of the version distributed on Steam.

## Features

* Original zones
* Original guilds
* Main quests and side content
* New characters
* Drama events
* Lesson and school systems
* Weather and seasonal features
* Additional gameplay systems and utilities

## Requirements

* Elin
* CWL (Custom Whatever Loader)
* Mod Help

## Project Structure

This repository consists of two plugins.

### NKAcademy

The main gameplay plugin.

Responsible for:

* Zones
* Guilds
* Quests
* Characters
* Lessons
* Weather
* Seasonal systems
* Miscellaneous gameplay features

### NKDramaTrigger

An event and trigger framework used by NKAcademy.

Provides:

* Character triggers
* Position triggers
* Zone triggers
* Key triggers
* Runtime databases
* Event evaluation systems

Both assemblies are independent BepInEx plugins and are intended to work together.
